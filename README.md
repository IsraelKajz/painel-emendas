# Painel de Emendas Parlamentares — deputados & partidos

Painel estático (arquivo único) sobre emendas parlamentares individuais:
quem destina, para onde vai, alinhamento com o partido do prefeito (TSE),
emendas Pix com objeto declarado (Transferegov) e sinais de atenção.

- **Painel**: [index.html](https://israelkajz.github.io/painel-emendas/)
- **Nota metodológica**: [NOTA_METODOLOGICA.html](https://israelkajz.github.io/painel-emendas/NOTA_METODOLOGICA.html)

Fontes: Portal da Transparência/CGU, TSE, Transferegov, STN, SIOP, IBGE, BCB,
Câmara e Senado — todas públicas e oficiais. Os sinais exibidos são **triagem
para escrutínio público, não prova de irregularidade** (limiares documentados
na metodologia). Pipeline de geração: repositório local `DadosEmendas`.
